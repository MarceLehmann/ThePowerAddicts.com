## 🚀 Getting Started

### Quick Development Setup

```bash
# Windows users:
dev.bat setup      # Install dependencies
dev.bat serve      # Start development server

# Mac/Linux users:
./dev.sh setup     # Install dependencies
./dev.sh serve     # Start development server
```

### Production Deployment

```bash
# Build for production
dev.bat build      # Windows
./dev.sh build     # Mac/Linux

# Deploy _site/ folder to your hosting provider
```
